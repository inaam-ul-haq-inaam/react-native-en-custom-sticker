# react-native-en-custom-sticker

custom steaker plugin react native 

## Installation


```sh
npm install react-native-en-custom-sticker
```


## Usage


```js
import { EnCustomStickerView } from "react-native-en-custom-sticker";

// ...

<EnCustomStickerView color="tomato" />
```
